# Google Meet Attendance
With this chrome extension you can easily record which student attendant your class.

#Details
Currently the extension is not on CWS (chrome web store). To test the extension you need to download all the files, go to extensions and set it up in a developer mode.

#How it works
First, join any meeting. Press on the icon and popup will show up, in ther popup you can create class, edit, or delete class.
