# Google Meet Attendance
With this chrome extension you can easily record which student attendant your class.

# Link
https://chrome.google.com/webstore/detail/google-meet-attendance/opkclbdjfkagpgoflleeeklgljfgfglk?hl=ru&authuser=0

# Description
Virtual classes have become the norm. 
Teachers have to manually check student participation. 
Google-Meet-Attendance is a convenient extension which helps teachers take student attendance: create a class with student names and schedule starting time. The extension will automatically sort students in to Early 🟢, Late 🟡, or Gone 🔴.

# How it works
## Creating a New Class
      1. Open Class Settings. 
      2. Click the green “Add Class” button. This will bring up Edit Options.
      3. Fill in the class name 
      4. Fill in the student names:
      a. Automatically (Recommended):
      i. Make sure that all your students in the class are present on the Google Meet.
      ii. Click on “Add All Current Students”. We recommend this option because it will save your time. After this, Attendance Manager will automatically sort students.
           b. Manually:
               i. Write down the student’s Google Account names in the box. To delete names, click on the X button next to the name. 
      5. Press on the green “Save” button.

![Add Class presentation](https://github.com/Hereugo/Google-Meet-Attendance/blob/main/Github/Add-Class.gif)

## Edit Class
   1. Open Class Settings.
   2. Press the white “Edit Class” button 
   3. Choose the class you want to edit from the list.
   4. Press the green “Edit” button. This will bring up Edit Options. See “How to Create a New Class?”
   5. Click the green “Save” button to save your changes.

## Delete Class
   1. Open Class Settings. 
   2. Click the red “Delete Class” button 
   3. Choose the class you want to delete from the list.
   4. Press the red “Delete” button. Remember, you cannot recover deleted classes.
    
![Select Class presentation](https://github.com/Hereugo/Google-Meet-Attendance/blob/main/Github/Edit-Del-Class.png)

# Attendance Manager
Click the “Show Everyone” button to open Attendance Manager. 
Attendance Manager does 3 things: sort students, choose a class, and schedule time. 
To update the Attendance Manager with new class changes, just close and reopen “Show Everyone”.
<img align="right" src="https://github.com/Hereugo/Google-Meet-Attendance/blob/main/Github/View-Attendance.png">
## How to Choose a Class?
   Choose a class you want to see in the class drop-down list.
## How to Set Time?
   Write the time your class will begin in “Period Start”. Attendance Manager will sort students based on “Period Start”.
## How to Sort Students?
   Choose an order option in the drop-down list. This will automatically say which students were early, late, or gone.
## How to Export a Student List?
   Choose an order option.
   Click Export. This will download a sorted list of students, such as a List of All Missing Students. The list is saved in the Downloads folder.
